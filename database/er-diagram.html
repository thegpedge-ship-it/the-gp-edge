Production schema for the GP Edge RACGP exam-prep & clinical-tools platform. Every multi-valued attribute is a child table (no arrays/CSV), and all aggregate numbers (attempts, mastery, usage, streaks) are computed via views — so no fact is stored twice. Binary files (question images, library PDFs, avatars, invoices) live in Cloudflare R2; only their metadata + object keys sit in the files table.
